Items


Materials:

Stick

  • In Game Desc: A BASIC CRAFTING MATERIAL
  • Can be obtained by breaking sticks on the ground, sticks can be broken with the (NONE) tool.

Small Rock

  • In Game Desc: A BASIC CRAFTING MATERIAL
  • Can be obtained by breaking small rocks on the ground, small rocks can be broken with the (NONE) tool.

Wood

  • In Game Desc: A CRAFTING MATERIAL. USEFUL FOR BUILDING
  • Can be obtained by cutting up logs from a fallen tree using an axe.

Crops:

Wheat

  • In Game Desc: A BASIC CROP. DROPS FROM HARVESTING WHEAT
  • Can be obtained by harvesting fully grown wheat plants with a sickle.

Seeds: Seeds do not go to the items tab but instead are stored in Seed Bags which is a tool.

Wheat Seeds

  • Can be obtained by harvesting fully grown wheat plants with a sickle.

Meat:

Raw Pork

  • In Game Desc: RAW FOOD. DROPS FROM PIGS
  • Can be obtained by killing a pig with any weapon.

Raw Beef

  • In Game Desc: RAW FOOD. DROPS FROM COWS
  • Can be obtained by killing a cow with any weapon.
